# Task Tracker CLI

---
A simple CLI app to track your tasks and manage your to-do list using Python.
Example solution to the project [Task Tracker CLI](https://roadmap.sh/projects/task-tracker) challenge from [roadmap.sh](https://roadmap.sh)

## Reuirements

---
> - Python 3.10+

## Feature and Usage

---
> - add task
> ```commandline
> python3 task_cli.py add 'Learn pyhon'
> ```

> - delete task
> ```commandline
> python3 task_cli.py delete 1
> ```

> - update task
> ```commandline
> python3 task_cli.py update 2 'Learn python again' 
> ```

> - mark task
> ```commandline
> python3 task_cli.py mark-in-progress 1
>
> python3 task_cli.py mark-done 2
> ```

> - list task
> ```commandline
> python3 task_cli.py list
> 
> python3 task_cli.py list todo
> 
> python3 task_cli.py list in-progress
> 
> python3 task_cli.py list done
> ```
